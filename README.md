🇨🇮 MonBus CI - L'Application du Peuple
MonBus CI est une solution mobile innovante conçue pour transformer l'expérience du transport urbain en Côte d'Ivoire. Ce prototype présente une interface moderne, fluide et adaptée aux réalités locales.
🚀 Vision du Projet
L'objectif est de digitaliser le quotidien des usagers des bus (SOTRA et autres) en intégrant des services communautaires et financiers au sein d'une seule interface.
✨ Fonctionnalités Clés
• 🚌 Suivi des Lignes : Consultation en temps réel des lignes majeures (Ligne 11, 81, 210, etc.) avec estimation des temps d'attente.
• 🤝 Solidarité Live : Un espace social où les usagers partagent des infos en direct (embouteillages, bus pleins, incidents) pour s'entraider.
• 💰 Mon Gaba : Un coffre-fort numérique sécurisé permettant aux usagers d'épargner de petites sommes quotidiennement pour leurs futurs tickets ou abonnements.
📱 Expérience Utilisateur
• Design SOTRA : Couleurs officielles (Orange & Vert) pour une reconnaissance immédiate.
• Mode Web App : Optimisé pour une installation directe sur l'écran d'accueil du smartphone sans passer par l'App Store ou le Play Store.
• Partenariats Intégrés : Espaces publicitaires natifs pour des marques comme Canal+, Wave CI et Nido.
🛠️ Stack Technique
• Frontend : HTML5 / Tailwind CSS (pour un design ultra-rapide).
• Logique : JavaScript (ES6+).
• Déploiement : Vercel (Haute disponibilité).
Développé avec passion pour l'innovation numérique ivoirienne.
